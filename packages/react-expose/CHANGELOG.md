# react-mission-control

## 1.0.0

### Features

- MCProvider, MCWrapper, MCTrigger components
- useMCActions, useIsMCActive hooks
- Mobile Safari-like tab switcher support
- Keyboard shortcut activation (configurable)
- CSS variables for customization
- Full TypeScript support
- SSR compatible (Next.js, Remix)
- Backward compatible aliases for Expose* names
